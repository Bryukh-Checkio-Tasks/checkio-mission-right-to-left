Einer der Roboter hat eine einfache Aufgabe bekommen: Er soll aus einer Folge einzelner Wörter einen Satz bilden, der dann als Wegbeschreibung dienen soll, die um das Raumschiff herum führt. Allerdings ist dieser Roboter Linkshänder und obendrein ein Spaβvogel. Er kann es einfach nicht lassen, seine rechtshändigen Kollegen zu veräppeln.

Gegeben wird eine Folge von einzelnen Worten als Strings. Du sollst aus diesen einzelnen Worten einen zusammenhängenden String bilden, in welchem die einzelnen Worte durch Kommata getrennt sind. Um die rechtshändigen Roboter zu veräppeln, sollst Du au√üerdem jedes "right" durch "left" ersetzen, auch wenn das "right" nur teil eines Wortes ist. Alle Worte sind klein geschrieben!

Eingabe: Eine Folge von Worten als Tuple (unicode).

Ausgabe: Den Text als String (einzelne Worte durch Kommata getrennt).

Beispiele: 

left_join(("left", "right", "left", "stop")) == "left,left,left,stop"
left_join(("bright aright", "ok")) == "bleft aleft,ok"
left_join(("brightness wright",)) == "bleftness wleft"
left_join(("enough", "jokes")) == "enough,jokes"

Wozu das gut ist: Diese Aufgabe gibt ein einfaches Beispiel für den Umgang mit Strings und Folgen.
